# Module 6 - Spring Data JPA and Hibernate Exercises

This module provides a beginner-friendly Spring Boot application for practicing:

- Spring Boot + Spring Data JPA setup
- JPA entity mapping
- Repository query methods
- REST CRUD endpoints
- In-memory H2 database usage

## Run the application

```bash
mvn spring-boot:run
```

## Main endpoints

- GET /employees
- GET /employees/{id}
- POST /employees
- DELETE /employees/{id}
- GET /employees/department/{department}
- GET /employees/search?lastName=Smith
